Although all of the courses I have taken during my undergraduate and graduate education have contributed to the skills and knowledge I have now, there are a few that I would like to highlight as they are especially revelant to my career interests.

SI 618: Data Manipulation & Analysis

In this course we learn how to leverage a variety of Python modules in order to process and analyze data with the goal of producing meaningful insights. Topics covered include working with messy data, exploratory data analysis, model fitting, machine learning techniques including dimensionality reduction, clustering and classification, natural language processing and big data processing.

LHS 712: Natural Langugage Processing on Health Data

This course covers techniques for extracting information from unstructured data present in health documents including Electronic Health Records and clinical narratives. As part of the course, students participate in a semester-long project in which natural language processing techniques are applied to a real-world health challenge.

SI 507: Intermediate Programming

This course builds upon previous introductory Python courses in order to provide students with a strong foundation in programming. Topics covered include testing, Git and Github, APIs, caching, web scraping and crawling, plotting, databases and flask.

SI 539: Design of Complex Websites

This course provides a hands-on approach to learning responsive, accessible front-end programming for Web Design. Topics covered include HTML5, CSS3, JavaScript, jQuery, additional frameworks, and the POUR design principles of accessible design.
